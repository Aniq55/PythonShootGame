# Shoot 'em All

A simple shooting game written in Python.


# Requirements

1. Python 2.7
2. Python-Pygame

# Installing Pygame

```bash
$ sudo -H pip install pygame
```

# Running the Game
  
```bash
$ python mainGame.py
```
